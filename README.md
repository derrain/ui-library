# nued-ui
A Nuxt 3 based simple UI component library

## Live Demo
[NUED Demo Page](https://www.nued.design/)

## How to Use
`npm i nued-ui`

## Dependencies
`npm install --save-dev @iconify/vue`

## `Example Usage`
```
<script setup>
  import { NuedIcon } from 'nued-ui';
</script>

<template>
  <NuedUI name="alarm" size="large" />
</template>
```
